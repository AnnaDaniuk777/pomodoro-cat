# Скриншоты для README

Сюда нужно положить PNG-скриншоты приложения. На них ссылается корневой [`README.md`](../README.md):

| Файл | Что на нём |
|---|---|
| `main.png` | Главный экран: кот, таймер, кнопки, нижняя навигация |
| `settings.png` | Экран настроек |
| `player.png` | Музыкальный плеер с плейлистом |
| `todo.png` | Список задач |
| `widget-timer.png` | Плавающий виджет таймера (свёрнутое окно) |
| `mini-player.png` | Мини-плеер (свёрнутое окно) |

Виджет, мини-плеер и трей видно только в Electron-сборке (`npm run dev`), в браузерной версии их нет. Их скриншоть из реального окна приложения.
